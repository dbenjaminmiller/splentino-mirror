# Splentino

The Splentino font family is a free and open-source implementation of the "Plaintin" typeface, created by Ben Byram-Wigfield for inclusion with Dorico 6. The fonts were released under the SIL Open Font License (OFL) version 1.1 and are packaged here for easy redistribution.

* [Information on the creation of Splentino](https://blog.dorico.com/2025/06/the-splentino-typeface-reviving-plantin-for-musical-scores/)
